# gaming_webpage

This is my gaming webpage. It is a webpage that I am working on to practice front-end web development. The plan for this project is to use HTML, CSS, and JS technologies to create a website which features reviews that I give games, gives a list of games that I have played, and a form to suggest games.

# The style

The page will feature a header, main, footer style, and will remain consistent across all pages. The header will feature links to the other pages. The main will feature the actual content of the page. And the footer will just feature some socials and contacts.

# The Entry

The entry point to my website will feature a biography of me, this project is meant to more of a "portfolio" type project and is meant to lead as a project to build up my skills so that I can eventually create a website that houses links to all of my projects.

# Reviewed games

The "Read a Review" page will list games that I have reviewed, the games I have reviewed are listed as cards with the cover arts of the games. There will also be a search box where someone can filter for a specific game they are looking for, which in the current rendition and until I get to the JS part of the project will just be for looks.

# Games I have played

This page will list all of the games I have played in steam. The plan is to use a similar idea to the reviews will card type images of the games I have played, and a search bar to search for a game.

# Suggest a Game

This page will house a form for someone to suggest a game, this will go know just a message of success, if time permits I think it would be fun to set this to go to my email. Once submitted I want to use the list of games that I have played to verify that the suggested game would be a new game to me.
